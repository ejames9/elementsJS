Installation

elements can be installed using either bower or npm package managers. You may also include the cdn link in your html.

The cdn link:

<script src="https://cdn.rawgit.com/ejames9/elementsJS/gh-pages/index.js"></script>

Why?

Q: Why not just use jQuery?

A: Why not just not use jQuery? This library is for developers who, like me, have found themselves working with a small subset of jQuery's VAST library most of the time, and that the resulting code was less than aesthetically pleasant. What started as a few replacement convenience alias-functions, ballooned into a full-blown DOM manipulation library with an optional new element syntax. There are quite a few convenient functions/methods that make writing JavaScript faster, easier and more fun. It may even beautify your code a bit! Try it.. if you like it, I'll refund all of the dollar signs you would've needlessly piled into all of your future code! The library also includes:

  • HTTP requests...
  • Event handling...
  • Loggers...
  • Utility functions and more.....

elements Syntax.

If you use a build system like gulp or grunt, you can try out the optional element syntax! It's guaranteed to melt your face! See below:


  <'#list-items, #sideNav ul'/>
                 .every((element)=> {
                       element
                           .bgColor('#FFF')
                           .color('blue')
                           .fontFamily('hack');
                 });

  click(<'#submit'>, ()=> {
    handleClick();
  });
